## 开启 ssh
参考[官方文档](https://wiki.deepin.org/en/System_Management/Service_Management/SSH_service)，注意此系统的 ssh 服务直接叫 ssh 不是 sshd。
```bash
sudo systemctl start ssh
# sudo apt-get install openssh-server
# sudo /etc/init.d/ssh start
```
