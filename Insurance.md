保险调研

优先级：医疗险 → 重疾险 → 定期寿险 → 意外险。


| 险种     | 主要特点                                                             |     |
| -------- | -------------------------------------------------------------------- | --- |
| 医疗险   | 去医院保险公司给医药费，即治病有钱付。                               |     |
| 重疾险   | 患重大疾病没法搬砖了，可得到一笔钱，颐享天年。                       |     |
| 定期寿险 | 『自己牺牲，照亮家人』如果自己挂（死或残）了，可得到一笔钱给家人用。 |     |
| 意外险   | 发生意外了，能得到一笔钱。                                           |     |
