## 安装
可以[下载](https://rime.im/download/)安装，也可以通过命令安装
```
brew install --cask squirrel
```

安装完成后需要退出系统登录，然后才能在键盘配置里增加输入法.

输入文字后使用『Ctrl』+『~』进行切换输入法方案。如双拼和简拼。
