手机打开 nfc 开关

下载软件[MifareClassicTool](https://xiakele.lanzouq.com/i67Ccygn7rc)

点击编辑/新建密钥文件,右上角加号,随便起个名字如：my.key

粘贴下面的内容
```bash
# chuncheng-360b4cfe-key
E4C2245A84AA
# defulat key
ffffffffffff
# others 
```
